== Description ==

PagoFacil Checkout es una de las principales pasarelas de pago de Bolivia el cual tiene integrado multiples tipos de cobros tales como tarjeta de débito o credito , TigoMoney , QR simple .

* Crea ordenes de pago 
* Actualiza los estados de su pedido de manera simultanea

= Usage =

PagoFacil Checkout Payment Gateway es visible para sus clientes como un botón de pago, el cual dará acceso al usuario a nuestra plataforma para realizar el pago, una vez finalizado el pago, el usuario sera redirigido a su tienda 

== Installation ==

Para poder utilizar nuestra pasarela de pagos necesitas tener unos credenciales, para obtenerlos por favor contactanos al siguiente correo **atencion.cliente@pagofacil.com.bo**

** Cualquier duda o problema que presentes puedes realizar tus consultas al : atencion.cliente@pagofacil.com.bo**

En el Panel de administración de Wordpress  :

1. Vaya  a  **WooCommerce** -> ** sección Configuracion **

2. Busca  **Pagos** desplácese  hacia abajo hasta encontrar  **"PagoFacil Checkout  "** 

3. Debes  **Activar ** el método de pago 

4. Luego entrar a Gestionar 

5. Se te aparecera  un formulario donde ingresaras los credenciales que se te han proporcionado 

6. Luego dar click en **Guardar Cambios**
